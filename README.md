## 项目简介

本项目基于 **STM32F767** 开发板和 **STM32Cube.AI** 工具，实现了一个嵌入式手写数字识别系统。

项目采用提供的预训练模型 **trained_lstm.tflite** 作为手写数字识别网络，并利用 STM32Cube.AI 工具完成模型解析、代码生成以及嵌入式部署。根据课题要求，项目工作重点在于人工智能模型在 STM32H747I-DISCO 开发板上的部署与推理实现，包括模型转换、嵌入式集成、串口通信以及识别结果展示等关键环节。通过构建完整的手写数字识别系统，验证了神经网络模型在资源受限嵌入式平台上的运行能力和应用效果。

用户可以在电脑端上位机中绘制 `0` 至 `9` 的手写数字，并通过串口将图像数据发送至 STM32 开发板。开发板接收数据后，调用部署在单片机上的神经网络模型完成推理，随后将识别结果返回电脑端显示。

## 主要功能

* 电脑端手写数字输入；
* 上位机与 STM32 串口通信；
* STM32 端神经网络推理；
* 手写数字识别结果返回与显示；
* 基于 STM32Cube.AI 的神经网络模型部署与运行。

## 文件说明

### 1. recognition_ide.rar

项目完整工程源码压缩包，包含：

* STM32CubeMX 配置文件；
* STM32Cube.AI 模型部署代码；
* STM32CubeIDE 工程文件；
* 用户实现的串口通信代码；
* 神经网络推理相关代码。

该工程可用于项目复现、编译以及程序烧录。

### 2. trained_lstm.tflite

本项目采用的预训练手写数字识别模型文件。

利用 STM32Cube.AI 工具完成模型解析、代码生成以及嵌入式部署，实现神经网络在 STM32H747I-DISCO 开发板上的推理运行。

### 3. 上位机程序

本项目所使用的上位机程序基于开源项目 **HandWriteApp** 实现，其源码由开源社区提供。

该程序提供以下功能：

* 手写输入区域；
* 串口配置；
* 数据发送；
* 识别结果显示。

能够满足手写数字识别系统的人机交互需求。

上位机测试软件地址：

https://github.com/colin2135/HandWriteApp.git

### 4. 基于STM32与X-CUBE-AI的手写数字识别系统设计.pdf

项目报告文件，详细介绍系统设计方案、模型部署流程、软件实现以及实验结果分析。

### 5. 演示视频

包含：

* 外摄版演示视频；
* 录屏版演示视频。

视频展示了输入数字 `0~9` 共 10 个数字时的识别过程与识别结果。

## 使用说明

1. 解压 `recognition_ide.rar` 工程文件；
2. 使用 STM32CubeIDE 打开工程并完成编译；
3. 将程序烧录至 STM32H747I-DISCO 开发板；
4. 打开电脑端 HandWriteApp 上位机程序；
5. 选择对应串口和波特率；
6. 在手写区域绘制数字并点击 `Send`；
7. 开发板接收数据并执行神经网络推理；
8. 在上位机界面查看识别结果。
