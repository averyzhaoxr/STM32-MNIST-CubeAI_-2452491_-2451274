## 项目简介

本项目基于 **STM32F767** 开发板和 **STM32Cube.AI** 工具，实现了一个嵌入式手写数字识别系统。

用户可以在电脑端上位机中绘制 `0` 至 `9` 的手写数字，并通过串口将图像数据发送至 STM32 开发板。开发板接收数据后，调用部署在单片机上的神经网络模型完成推理，随后将识别结果返回电脑端显示。

## 主要功能

* 电脑端手写数字输入；
* 上位机与 STM32 串口通信；
* STM32 端神经网络推理；
* 手写数字识别结果返回与显示。

## 使用说明

1. 使用 STM32CubeIDE 编译并烧录程序；
2. 打开电脑端 HandWriteApp 上位机；
3. 选择对应串口和波特率；
4. 绘制数字并点击 `Send`；
5. 在界面中查看识别结果。
